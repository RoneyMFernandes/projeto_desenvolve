O Caminho para a Iluminação

Espalhados por uma estrada escura e sombria estão os restos do passado: tags específicas de navegador, DOMs incompatíveis, suporte quebrado a CSS e navegadores abandonados.

Devemos limpar a mente do passado. A iluminação da web foi alcançada graças aos incansáveis esforços de pessoas como o W3C, o WaSP e os principais desenvolvedores de navegadores.

O CSS Zen Garden convida você a relaxar e meditar sobre as importantes lições dos mestres. Comece a enxergar com clareza. Aprenda a usar as técnicas consagradas de maneira nova e revigorante. Torne-se um com a web.

Do que se trata?

Existe uma necessidade contínua de mostrar o poder do CSS. O Zen Garden tem como objetivo entusiasmar, inspirar e encorajar a participação. Para começar, veja alguns dos designs existentes na lista. Clicar em qualquer um irá carregar a folha de estilo nesta mesma página. O HTML permanece o mesmo, a única coisa que muda é o arquivo CSS externo. Sim, é verdade.

O CSS permite controle completo e total sobre o estilo de um documento hipertexto. A única maneira de ilustrar isso de forma empolgante é demonstrando o que realmente pode ser feito, uma vez que as rédeas estão nas mãos de quem é capaz de criar beleza a partir da estrutura. Designers e programadores contribuíram para a beleza da web; sempre podemos ir além.

Participação

O design visual forte sempre foi nosso foco. Você está modificando esta página, portanto habilidades sólidas com CSS também são necessárias, mas os arquivos de exemplo estão bem comentados, de modo que até mesmo iniciantes em CSS podem usá-los como ponto de partida. Consulte o Guia de Recursos CSS para tutoriais avançados e dicas de uso.

Você pode modificar a folha de estilo da maneira que quiser, mas não o HTML. Isso pode parecer difícil no início se você nunca trabalhou assim antes, mas siga os links listados para aprender mais e use os arquivos de exemplo como guia.

Baixe o HTML de exemplo e o CSS para trabalhar localmente. Quando terminar sua obra-prima (e por favor, não envie trabalhos inacabados), envie seu arquivo CSS para um servidor web sob seu controle. Envie-nos um link para um arquivo compactado contendo esse arquivo e todos os recursos associados. Se o escolhermos, faremos o download e hospedaremos em nosso servidor.

Benefícios

Por que participar? Por reconhecimento, inspiração e como um recurso para mostrar às pessoas como o CSS pode ser realmente incrível. Este site serve igualmente como inspiração para os que trabalham na web hoje, ferramenta de aprendizado para os que o farão amanhã e uma galeria de técnicas futuras pelas quais todos podemos esperar.

Requisitos

Quando possível, gostaríamos de ver o uso predominante de CSS 1 e 2. O uso de CSS 3 e 4 deve ser limitado a elementos amplamente suportados ou com fallback adequado. O CSS Zen Garden é sobre CSS funcional e prático, e não os truques mais recentes visíveis apenas por 2% do público. O único requisito real é que seu CSS seja válido.

Felizmente, projetar dessa forma mostra como os navegadores modernos implementaram bem o CSS. Seguindo as diretrizes, você deve ver resultados consistentes na maioria dos navegadores modernos. Com a grande quantidade de dispositivos disponíveis — especialmente móveis — layouts pixel-perfeitos nem sempre são possíveis. Está tudo bem, mas teste em tantos navegadores quanto puder. Seu design deve funcionar no mínimo no IE9+ e nas últimas versões do Chrome, Firefox, iOS e Android (que cobrem mais de 90% da população).

Pedimos que você envie obras originais. Respeite as leis de direitos autorais. Evite conteúdo ofensivo e tente incorporar temas visuais únicos e interessantes ao seu trabalho. Já passamos do ponto de precisar de mais um design relacionado a jardim.

Este é um exercício de aprendizado tanto quanto uma demonstração. Você mantém todos os direitos autorais sobre suas imagens (com exceções limitadas, veja as diretrizes de envio), mas pedimos que você libere seu CSS sob uma licença Creative Commons idêntica à usada neste site para que outros possam aprender com seu trabalho.
